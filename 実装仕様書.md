# Comic Viewer 実装仕様書（機能チェックリスト）

- 更新日: 2026-07-03
- 詳細仕様は [機能設計書.md](機能設計書.md) を参照

---

## ✅ 実装済み機能

### 本棚
- [x] フォルダ / ファイル登録（ダイアログから追加）
- [x] ドラッグ＆ドロップ登録（エクスプローラーからドロップで追加）
- [x] サムネイル付きグリッド表示（木目調デザイン・棚板描画）
- [x] フォルダ階層ブラウズ（ホーム / 上へ）
- [x] キーワード検索（サブフォルダ再帰）
- [x] 最後に開いた場所・スクロール位置の復元
- [x] しおり状態バッジ（赤=読みかけ / 青=読了）
- [x] ページキャッシュ済みフラグ表示
- [x] 閲覧履歴（最近読んだ本30件をツールバーの履歴メニューから開く）
- [x] ウィンドウ表示ロード（スクロール位置±50件のみロード、省メモリ）
- [x] グリッド幅の動的調整（右側の無駄な余白なし）

### ビューア
- [x] 1ページ表示 / 見開き2ページ表示
- [x] 読む方向切替（右→左 / 左→右)
- [x] ズーム（ホイール / キー）
- [x] フィットモード（全体 / 横幅 / 縦幅）
- [x] ページ回転（90°単位、ページごとに永続化）
- [x] しおり（ラベル付き・複数可）
- [x] サムネイルストリップ（下部ページ一覧）
- [x] スライドショー（秒数・終了時動作・切替エフェクト設定可）
- [x] 読書進捗の保存・復元
- [x] 全画面表示
- [x] 埋め込み / 独立ウィンドウの切替（設定）
- [x] 次の本 / 前の本へ移動
- [x] ページ指定ジャンプ

### 対応フォーマット
- [x] ZIP / CBZ（標準ライブラリ・ストリーミング展開）
- [x] RAR / CBR（libarchive → 7-Zip / unar フォールバック）
- [x] 7z / CB7（libarchive → 7-Zip / unar フォールバック）
- [x] PDF（pypdfium2 レンダリング・1ページずつ順次表示・ページキャッシュ対応）
- [x] アーカイブ内 PDF（ZIP / RAR / 7z 内に画像がなく PDF がある場合に展開。画像混在時は画像優先）
- [x] 画像フォルダ（jpg / jpeg / png / webp / gif / bmp）

### 基盤・非機能
- [x] ページキャッシュ（上限MB設定・LRU削除・mtime検証）
- [x] サムネイルキャッシュ（失敗プレースホルダー付き）
- [x] 非同期ワーカー（スキャン / サムネイル / 検索 / ページ読み込み）
- [x] スレッド安全停止（terminate不使用・ゾンビスレッド管理）
- [x] メモリ最適化（2000冊規模で約200MB以内）
- [x] ネットワークドライブ対応（未接続時のエラー抑制）
- [x] ZIP内ファイル名の文字コード自動判別（CP932 / UTF-8）
- [x] 多言語対応（日本語 / 英語）
- [x] Windows / Linux 両対応（単一ソース）
- [x] Windowsインストーラー / アンインストーラー（PyInstaller）
- [x] アプリアイコン（EXE埋め込み・タスクバー）
- [x] 統一デザインのツールバー / メニューアイコン（黒線＋丸囲みSVG）
- [x] AV誤検知対策（UPX無効化）
