## ✅ 実装済み機能

### 本棚
- [x] フォルダ / ファイル登録（ダイアログから追加）
- [x] ドラッグ＆ドロップ登録（エクスプローラーからドロップで追加）
- [x] サムネイル付きグリッド表示（木目調デザイン・棚板描画）
- [x] フォルダ階層ブラウズ（ホーム / 上へ）
- [x] キーワード検索（サブフォルダ再帰）
- [x] 最後に開いた場所・スクロール位置の復元
- [x] しおり状態バッジ（赤=読みかけ / 青=読了）
- [x] ページキャッシュ済みフラグ表示
- [x] 閲覧履歴（最近読んだ本30件をツールバーの履歴メニューから開く）
- [x] ウィンドウ表示ロード（スクロール位置±50件のみロード、省メモリ）
- [x] グリッド幅の動的調整（右側の無駄な余白なし）
- [x] 並び替え（名前順 / 追加日順 / 最終閲覧順 / 手動。ツールバーの並び順メニュー）
- [x] 手動配置（並び順=手動でドラッグして好きな段へ配置。フォルダごとに保存・空の段にも棚板を描画。
      実装方式: Qtの Free movement（自由なピクセル位置指定）はウィンドウリサイズのたびに
      Qt内部で位置が再計算されてしまう問題が何度も再発したため採用せず、他の並び順と
      全く同じ Movement.Static + ResizeMode.Adjust（実績のある自動整列）を使用。
      「手動配置」は見た目の位置ではなく**モデル内の行の並び順**そのもので表現し、
      空いている段は非表示のプレースホルダー行で埋める（末尾にも空席を数段確保し、
      「何もない本棚の場所」へドロップできる）。並べ替えのドラッグは Qt の内部ドラッグ
      （startDrag/QDrag.exec）が環境依存で不安定だったため使わず、mousePress/Move/Release
      による自前実装（押下行を記録し、離した位置の行と `_swap_rows` で入れ替え）。
      リサイズしても重ならず・震えない。ドロップ後は選択・カレントをクリアして誤フォーカスを
      防止。並びが変わるたびに全アイテムの(段,列)を丸ごと保存して決定的にする。並び順メニューに
      「この本棚の手動配置をリセット（名前順に戻す）」を用意）
- [x] 手動配置リセット（並び順メニューから手動配置を破棄して名前順に戻す）
- [x] 既読 / 未読フィルター（すべて / 未読 / 読みかけ / 読了）
- [x] シリーズ自動グループ化（同名巻数違いを「タイトル（全N冊）」にまとめ、ダブルクリックで巻一覧）
- [x] 読書統計（登録数・読了数・しおり数・直近7/30日に開いた冊数）
- [x] パンくずリスト（ヘッダーを階層ナビ化。🏠ホーム › 本 › … 各段クリックでその階層へジャンプ）
- [x] 選択アイテムのファイルパス表示（設定トグル show_filepath でON/OFF。ONで本棚下部にパス+ファイル名）
- [x] サムネイルサイズのクイック切替（ツールバー「表示」ボタンで 110/150/200/260px をワンクリック巡回。
      再スキャンせず表示範囲のみ再描画で高速）

### ビューア
- [x] 1ページ表示 / 見開き2ページ表示
- [x] 読む方向切替（右→左 / 左→右)
- [x] ズーム（ホイール / キー）
- [x] フィットモード（全体 / 横幅 / 縦幅）
- [x] ページ回転（90°単位、ページごとに永続化）
- [x] しおり（ラベル付き・複数可）
- [x] サムネイルストリップ（下部ページ一覧）
- [x] スライドショー（秒数・終了時動作・切替エフェクト設定可）
- [x] 読書進捗の保存・復元
- [x] 全画面表示
- [x] 埋め込み / 独立ウィンドウの切替（設定）
- [x] 次の本 / 前の本へ移動
- [x] ページ指定ジャンプ
- [x] 横長ページの自動分割（1ページモード: 左右半分ずつ表示 / 見開きモード: 横長ページを単独全幅表示。読み方向対応・ON/OFF可）
- [x] 画像フィルタ（明るさ補正 50〜150% / グレースケール / 背景色 5色プリセット）
- [x] 余白自動カット（非白領域を検出してトリミング。全白ページはガード）
- [x] 縦スクロールモード（ウェブトゥーン対応。可視範囲±2画面のみロードで省メモリ）
- [x] キーカスタマイズ（設定画面から8アクションのキー割り当てを変更可）
- [x] ルーペ（Lキーでトグル、カーソル追従で原寸表示）
- [x] マウスジェスチャー（右ドラッグ: ←次ページ / →前ページ / ↑全画面 / ↓本棚へ。ドラッグなしは従来のメニュー表示）

### 対応フォーマット
- [x] ZIP / CBZ（標準ライブラリ・ストリーミング展開）
- [x] RAR / CBR（libarchive → 7-Zip / unar フォールバック）
- [x] 7z / CB7（libarchive → 7-Zip / unar フォールバック）
- [x] PDF（pypdfium2 レンダリング・1ページずつ順次表示・ページキャッシュ対応）
- [x] アーカイブ内 PDF（ZIP / RAR / 7z 内に画像がなく PDF がある場合に展開。画像混在時は画像優先）
- [x] 画像フォルダ（jpg / jpeg / png / webp / gif / bmp）

### 基盤・非機能
- [x] ページキャッシュ（上限MB設定・LRU削除・mtime検証）
- [x] サムネイルキャッシュ（失敗プレースホルダー付き）
- [x] 非同期ワーカー（スキャン / サムネイル / 検索 / ページ読み込み）
- [x] スレッド安全停止（terminate不使用・ゾンビスレッド管理）
- [x] メモリ最適化（2000冊規模で約200MB以内）
- [x] ネットワークドライブ対応（未接続時のエラー抑制）
- [x] ZIP内ファイル名の文字コード自動判別（CP932 / UTF-8）
- [x] 多言語対応（日本語 / 英語）
- [x] Windows / Linux 両対応（単一ソース）
- [x] Windowsインストーラー / アンインストーラー（PyInstaller）
- [x] アプリアイコン（EXE埋め込み・タスクバー）
- [x] 統一デザインのツールバー / メニューアイコン（黒線＋丸囲みSVG）
- [x] ヘルプガイド（本棚 / ビューア各モードの説明書。設定の左の？ボタンから表示・日英対応）
- [x] AV誤検知対策（UPX無効化）
